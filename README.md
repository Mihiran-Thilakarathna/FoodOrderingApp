# Food Ordering App - ICT3214 Group Project

## Project Overview
This is an Android mobile application developed for the **ICT3214 - Mobile Application Development** module. The application allows users to browse a food menu, add items to a cart, and place orders seamlessly. The system is built using **Java** and **SQLite** for local data storage, ensuring a smooth user experience without requiring an external payment gateway.

## Features
The application includes the following core functionalities as per the project guidelines:

* **User Authentication:** Secure Login and Registration system with password encryption.
* **Menu Management:** Users can view the available food menu items.
* **Cart System:** Ability to add food items to a cart.
* **Order Placement:** Users can confirm and place orders (simulation without payment gateway).
* **Data Privacy:** Logged-in users can only view their own order history and data.
* **Profile Management:** User logout functionality.

## Technologies Used
* **Language:** Java (Android SDK)
* **Database:** SQLite (Local Database)
* **IDE:** Android Studio
* **Version Control:** GitHub
